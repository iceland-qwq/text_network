
movie_comments 是各个电影的评论 
100000-small 是词向量模型 里面是常用中文100000词和他在词向量中的坐标位置 #这里没有放 有需要的联系我 
cleandata 是当时清理好的评论数据

model_keras_lstm 是训练好的神经网络模型 可以用来预测评论正负向

movie_config是当时100个喜剧电影的评分和名字 
nb_svm是使用CountVector训练的稀疏矩阵词向量 只能用于贝叶斯和向量机的训练 

network 就是神经网络模型的训练 主要是两个模型 一个是双向lstm 一个是一维cnn 
标签的加载 是使用post爬虫调用百度云api接口 对所要训练的数据进行一个标签的加载 
评论正相关 是使用训练好的神经网络模型预测各个电影的评论 然后计算出正向评论的比例 最后得出这个和评分正相关的信息 最后再定义了一个线性回归模型 训练这个关系 就可以根据一个电影正向评论的比例 来预测这个电影的真是评分情况 给系统推荐做一个参考
最后那个model.png是这个模型的结构图
